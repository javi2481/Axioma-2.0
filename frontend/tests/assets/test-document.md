OpenRAG Project Overview

OpenRAG is a platform designed to simplify the creation of agentic RAG systems.
It integrates seamlessly with various LLM providers and vector databases to provide a robust knowledge base for AI agents.

Key features include:
- Intelligent Document Ingestion: Support for multiple file formats including PDF, TXT, and Markdown.
- Vector Storage: Optimized retrieval using state-of-the-art vector similarity search.
- Agentic Workflows: Complex orchestration of tasks using advanced AI reasoning.
- Extensible Architecture: Easily add new components and integrations.
